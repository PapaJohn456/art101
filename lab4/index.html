Problems

What problems arose? How did you solve them?

The way I had the winner function detect the winner was by checking the list that had the player input first and then the computer input. It would if the player input was rock,paper or scissors and based off that answer it would check the second item in the list, computer input, to check the win conditions. One issue I had with this is that the list would exceed 2 items. The quick solution was to add an if statement that would check if the list exceeded 2 items in which it would reset it. This would then add another issue where it would do that, but because of the placement it would still exceed 2, 4 being the max. The solution was to move it where it would create 4 and only have the players input in the list. I also had an issue where my button wouldn't work which was because I didn't correctly name everything the same.
